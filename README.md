# Controle Estudantil :school_satchel:

*Descrição:*
Bot criado para fins acadêmicos, como proposta de trabalho semestral da disciplina Interação Humano Computador lecionada pelo Profº Me. Giuliano Bertoti em 2019, na instituição de ensino Fatec Professor Jessen Vidal (São José dos Campos).


*Objetivo:*
O Bot tem como objetivo auxiliar na organização escolar de seu usuário, fornecendo os horários com as matérias, nomes de professores e a sala em que ocorrerá a aula, de todos os dias da semana separadamente, dos cursos disponibilizados FATEC São José dos Campos, assim não sendo necessário o usuário conferir o cronograma em outros sites ou arquivos, facilitando e agilizando a sua organização em relação aos horários. 


*Materiais e Métodos:*
 - Python e suas bibliotecas. 

 - JSON um formado de representação de dados baseado na linguagem de programação Javascript
 
 - Telepot que ajuda você a criar aplicativos para a API Telegram Bot. Ele funciona no Python 2.7 e no Python 3. 
 
 - BotFather que é usado para criar novas contas de bots e gerenciar seus bots existentes.
